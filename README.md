# Some-experiments-of-STDM-submitted-for-IJEPES
This repository provides tools for critical node selection and basic transfer learning in power system applications. It includes guidance for using an existing open-source critical node selection method, along with example implementations for transferring models across different test systems (e.g., IEEE 33 to IEEE 37).
Ref: P. Xu, B. Wollenberg, Power system observability and optimal phasor measurement unit placement, University of Minnesota (2015).
